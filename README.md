# Cybersec-Learning

Bienvenue dans mon dépôt d'apprentissage en cybersécurité.  
Ici, je stocke mes exercices, scripts, labs et notes pour suivre ma progression.

## Contenu
- **network/** : Scans réseau, analyse de paquets, Wireshark, Nmap
- **pentesting/** : Exemples de tests de vulnérabilités sur VM
- **scripts/** : Scripts Python et Bash pour automatisation
- **notes/** : Commandes, concepts et bonnes pratiques
- **ctf_challenges/** : Challenges CTF pour s’entraîner

## Usage
- Cloner ce dépôt pour suivre mes exercices.
- Tester les scripts uniquement dans un environnement sûr (VM, sandbox)
